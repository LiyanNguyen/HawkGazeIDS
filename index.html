Hawk IDS

Hawk IDS is a lightweight, minimalist, intrusion detection system specifically made for mobile phones. It is a straightforward and easy to learn application that provides the basic functionalities for tracking and analyzing network packets

Use it anywhere you go.

Hawk IDS can be controlled and managed from your mobile phone on the go. Being a cloud-base application lets you perform your cyber security management virtually anywhere!

Keep track of your network.

Hawk IDS provides a real-time view and reportings of the network traffic coming into your device, so you can always take immediate action!

Quarantine network packets.

Hawk IDS can quarantine specified network traffics defined by you and notifies you immediately, so you are always in control of your cyber security!

Double check network packets.

Hawk IDS provides reports of network traffic that has entered your device up to 3 months, so you can always double check if there is a hole in your cyber security!
